## 任务：多功能环境监测站

### 需求

**复用已有的东西：
- OLED 显示
- DHT11 温湿度
- 按键交互

**新学的东西：**
- **ADC** — 读取光敏电阻/电位器的模拟值，显示光照强度（0~100%）
- **内部Flash** — 每 10 秒存一次温湿度数据到 Flash，最多存 128 组，掉电不丢，可以翻页查看历史

**功能描述：**

| 模式 | 说明 |
|------|------|
| 实时显示 | OLED 第一行：温度 + 湿度，第二行：光照强度 + 时间（从时钟项目里复用） |
| 曲线图 | 再按 PA0 切换到曲线模式，OLED 绘制最近 20 组温度变化折线图 |

**按键逻辑：**
- 短按 PA0 → 实时模式 → 曲线模式 → 实时模式（循环）
### 项目的学习点

| 知识点 | 难度 | 说明 |
|--------|:----:|------|
| ADC 采集 | ⭐⭐ | CubeMX 配置，HAL_ADC 轮询读取 |
| 内部 Flash 读写 | ⭐⭐⭐ | 按扇区擦除，写入数据，读回验证 |
| 环形缓冲区 | ⭐⭐ | 128 组数据的存取、覆盖旧数据 |
| OLED 画折线图 | ⭐⭐⭐ | 坐标变换，数据映射到像素点 |
| 多模式状态机 | ⭐⭐ | 比时钟多一个曲线模式 |

### 约束
1. **不要用全局变量满天飞** — 每个模块封装好自己的内部状态
2. **Flash 写入间隔 ≥ 10 秒** — 别把 Flash 写坏了（寿命 10 万次）
3. **曲线图要有坐标轴** — X 轴标序号，Y 轴标温度值
